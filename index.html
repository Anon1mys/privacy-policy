Privacy Policy

QuRe stores user data locally in the browser using Chrome storage, including templates, collections, and user settings.

The extension interacts with web page content only to insert user-created templates into input fields. It does not collect or transmit page content to external servers.

If the user chooses to sign in, QuRe uses Firebase (Google services) for authentication and account management. This may include processing user email and storing basic account data such as user ID and subscription status.

No personal data is sold or shared with third parties for advertising purposes.

All external communication is limited to essential services required for authentication and account-related functionality.

This policy may be updated in the future if new features such as analytics are introduced.
